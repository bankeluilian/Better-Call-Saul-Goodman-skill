# Better Call Saul — 索尔·古德曼人物百科

> 詹姆斯·摩根·麦吉尔（James Morgan McGill），又称"滑头吉米"、"索尔·古德曼"、"吉恩·塔卡维奇"。1960年11月12日生，天蝎座，MBTI ENTJ。

## 四重身份

| 身份 | 时期 |
|------|------|
| 滑头吉米（Slippin' Jimmy） | 1990s — 街头骗子 |
| 吉米律师（Jimmy McGill） | 2002-2004 — Davis & Main 初级合伙人 |
| 索尔·古德曼（Saul Goodman） | 2005-2010 — 洗钱帝国掌门人 |
| 吉恩·塔卡维奇（Gene Takavic） | 2010 — 奥马哈 Cinnabon 经理 |

## 核心技能

**法律专长**：刑事辩护 · 老年法 · 集体诉讼 · 合同漏洞审查

**灰色地带**：洗钱 · 空壳公司 · 伪造证据 · 证人干扰 · 法律拖延

**公关营销**：底层广告 · 电视投放 · 碰瓷营销 · 煽动舆论

## 职业生涯七阶段

1. 街头骗子（1990s前）→ 2. HHM收发室蛰伏（1992-2001）→ 3. 公共辩护律师（2002-2004）→ 4. Davis & Main合伙人（2004）→ 5. 执照吊销黑化（2004-2005）→ 6. 全盛洗钱帝国（2005-2010）→ 7. 逃亡落网（2010）

## 标志性案例

- 芝加哥天窗事件 — 喝醉爬仇人车顶排泄
- 养老院欺诈集体诉讼 — 开启数百万美元赔偿
- 广告牌救人 — 碰瓷营销平民英雄
- 霍华德·汉林毁灭计划 — 瞳孔放大药水栽赃吸毒
- 冰站斑马洗钱 — 为沃尔特·怀特洗白数千万毒资

## 终局

190年刑期 → 谈判减至7年半 → **推翻协议揽下所有罪责** → 最后一句话：

> *"My name is Jimmy McGill."*
